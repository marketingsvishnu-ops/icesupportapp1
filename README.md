# Play Store–Inspired WhatsApp Landing Page

## Edit these details

1. Open `script.js`
2. Replace `919999999999` with your WhatsApp number including country code.
3. Edit the app name, company name, description and customer reviews in `index.html`.
4. Replace placeholder initials/cards with your own logo and real screenshots.
5. Upload all files to Netlify, Vercel, GitHub Pages or your hosting panel.

## Important

- Use only genuine customer reviews.
- Keep the footer disclaimer visible.
- Do not present this page as an official Google Play Store listing.
